Minecrap CustomRP Setup Guide

Follow these steps to set up Custom Rich Presence for Discord:

  1. Download CustomRP.
  2. Open discord.exe, but before clicking Connect:
    • Go to Settings → Activity Privacy.
    • Ensure your settings are configured. (All settings are optional except for the ones specified below.)
  3. In CustomRP, ensure the following:
    • Application ID: 1326685266563436586
    • Image: Use the specified image to match the setup.
  4. Save your configuration and start CustomRP to display the "Minecrap Single Player" status.
